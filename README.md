# 电梯调度系统
同济大学2019年春操作系统课程第一次作业

##开发环境
Mac OS Mojave

##开发环境依赖

### Python3
[python3](https://www.python.org/getit/) 

### PyQt5

`pip3 install pyqt5`

### PyInstaller

`pip3 install pyinstaller`
 
## 测试环境
* macOS Mojave
* Windows 10

## 规则说明

* 电梯开门时间为1s


* 电梯运行通过一层楼间隔所需时间为0.4s
	
##使用说明

###1、 呼叫电梯
* **上行**

	点击左侧按钮中的⬆️按钮
* **下行**

	点击左侧按钮中的⬇️按钮
	
###2、 前往目标楼层  
点击所在电梯中的对应数字按钮
